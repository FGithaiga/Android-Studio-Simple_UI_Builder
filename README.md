# Simple UI Builder

Simple UI Builder is an Android application built with **Android Studio** that demonstrates how to design and configure user interfaces and widgets.  
It includes layouts, themes, and widget resources to showcase how UI components can be structured and styled in a modern Android app.

---

## 📱 Features
- Custom UI layouts (`activity_main.xml`, `widget_layout.xml`)
- Widget configuration (`widget_layout_info.xml`)
- Resource management (colors, strings, themes, styles, dimensions)
- Gradle build setup with Kotlin support
- Example unit and instrumented tests

## 🛠 Project Structure
- `app/src/main/java/` → Kotlin source code (`MainActivity.kt`, `widget_layout.kt`)
- `app/src/main/res/` → XML resources (layouts, drawables, mipmaps, values, themes, styles)
- `app/src/main/AndroidManifest.xml` → App manifest
- `gradle/` → Gradle wrapper files
- `build.gradle.kts` → Project build configuration
- `app/build.gradle.kts` → Module build configuration
